# NTCU-oop-system-auto-complete-tool
## Feature
An automated tool was developed for an OOP practice system designed for the **國立臺中教育大學資訊工程學系 114-2 資料庫程式設計** course, automating the AI-interactive questionnaire process and allowing for one-click submission of answers.

## Usage
```bash
bun install

vite build
```
Then you can see the built files in the `.dist` directory.

Or go to [release](https://github.com/moquan4543/NTCU-oop-system-auto-complete-tool/releases) to get the built version.
